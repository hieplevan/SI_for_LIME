Selective Inference for LIME
This repository provides a framework for performing Selective Inference (SI) on explanations generated by LIME (Local Interpretable Model-agnostic Explanations). By accounting for the selection bias inherent in LIME's feature selection process (e.g., via Lasso), this tool computes valid p-values and confidence intervals for the local explanations.

Project Structure
The project is organized into two main directories:

1. lime_si/ (Core Module)
This is the main package containing the logic for selective inference:

lime.py: Implementation of the LIME algorithm modified for SI compatibility.

parametric_lasso.py: Handles the Lasso optimization path and the geometry of the selection event.

compute_p_value.py: Functions to calculate the truncated Gaussian p-values based on the polyhedral lemma.

gen_data.py: Utilities for generating or preprocessing data for local explanation tasks.

util.py: Helper functions for matrix operations and statistical distributions.

2. example/ (Demonstration)
Ready-to-run scripts to understand how to apply SI to your models:

ex1_p_value.py: Basic example showing how to extract p-values for selected features.

ex2_pivot.py: Demonstrates the use of pivotal quantities in the inference process.

ex3_confidence_interval.py: Constructing valid selective confidence intervals for feature weights.
